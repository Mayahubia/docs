### Criando um Novo Aplicativo

Você precisa criar um **aplicativo do LinkedIn** para permitir que os usuários façam login no seu site usando suas contas do LinkedIn.\
Para criar um novo aplicativo, siga as etapas abaixo:

---

1. Acesse o site de desenvolvedores do LinkedIn: https://developer.linkedin.com/
2. Clique no botão **“Create App”** no banner.
3. Para criar o aplicativo corretamente, preencha as seções conforme explicado abaixo.

---

### Criar um Aplicativo

- **Application Name:** insira um nome para o seu aplicativo.
- **Company Name:** selecione uma empresa existente ou escolha **New Company**.
  - Se selecionar **New Company**, será solicitado que você insira o nome da empresa.
- **Description:** insira uma descrição para o seu aplicativo.
- **Business Email:** insira o seu endereço de e-mail comercial.
- **Legal Terms:** leia os **termos de serviço** e marque a caixa confirmando que leu e concorda com os termos.

Após preencher todas as informações, clique no botão **Create App** para concluir a criação do aplicativo.

## Produtos

Após clicar no botão **Create App**, você será redirecionado para a guia **Products**.

<Frame>
  ![Linkedin Products Tab](/linkedin-products-tab.avif)
</Frame>

Role para baixo na lista de produtos até encontrar **Sign In with LinkedIn using OpenID Connect** e clique no botão **Request Access**.

<Frame>
  ![LinkedIn OpenID Connect](/linkedin-openid-connect.avif)
</Frame>

Leia os termos e, se concordar, marque a **caixa de seleção** e prossiga clicando no botão **Request Access**..

<Frame>
  ![LinkedIn Request Access](/linkedin-request-acces.avif)
</Frame>

## Autenticação

Acesse a guia **Auth** e adicione as **URLs de Redirecionamento (Redirect URLs)**.

<Frame>
  ![LinkedIn Auth Tab](/linkedin-redirect-url.avif)
</Frame>

## Configurações do OAuth 2.0

Nas **URLs de redirecionamento**, basta inserir o endereço do seu site.\
**Exemplo:** `https://sss.com/login/linkedin-openid/callback`

<Frame>
  ![LinkedIn Redirect URI](/linkedin-redirect-url.avif)
</Frame>

## Escopos

Os **escopos do OAuth 2.0** aparecerão alguns segundos após você adicionar e atualizar as **URLs de redirecionamento** nas **Configurações do OAuth 2.0**.

<Frame>
  ![LinkedIn OAuth Scopes](/linkedin-oauth-scopes.avif)
