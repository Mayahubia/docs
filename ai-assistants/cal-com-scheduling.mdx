Ative sua assistente de IA para agendar compromissos e reuniões de forma automática com a integração [Cal.com](http://Cal.com). Essa funcionalidade permite que a IA verifique a disponibilidade e agende diretamente durante as chamadas.

<Warning>
  **Importante:**\
  Não adicione campos obrigatórios personalizados no evento do [Cal.com](http://Cal.com).\
  A integração só funciona com os campos padrão e o campo opcional de telefone. Campos personalizados obrigatórios fazem a integração falhar.
</Warning>

### Pré-requisitos

Antes de configurar a integração com o [Cal.com](http://Cal.com), você precisará de:

- Uma conta [Cal.com](http://Cal.com) (hospedada ou self-hosted)
- Um tipo de evento configurado no [Cal.com](http://Cal.com)
- Uma chave de API do Cal.com

### Etapas de Configuração

#### 1. Configuração no [Cal.com](http://Cal.com)

1. Sign up or log in to your [Cal.com](https://cal.com/) account
2. Crie um novo tipo de evento exclusivo para agendamentos da assistente de IA
   - Dê um nome como: **"Reunião com Assistente de IA"**
   - Configure duração, disponibilidade e outras opções
3. Após criar, anote o **ID do evento** (presente na URL)

   <img
     height="200"
     src="/resources/cal-event-id.png"
   />
4. **Gere uma chave de API:**
   - Vá em **Configurações do **[**Cal.com**](http://Cal.com)
   - Acesse **Developer Settings**
   - Crie uma nova chave de API
   - Copie e guarde sua chave com segurança

### 2. Informações de Contato do Cliente

**Configuração de E-mail**\
Para enviar convites de calendário aos clientes:

- Defina o campo de e-mail nas variáveis de entrada da assistente
- Ao criar o lead, envie o e-mail do cliente como variável
- O sistema usará esse e-mail automaticamente para o convite

**Número de Telefone no Evento**\
Para incluir o número de telefone do cliente nos eventos:

- Vá nas configurações do evento no [Cal.com](http://Cal.com)
- Clique em **“Adicionar uma pergunta”**

  <img
    height="200"
    src="/resources/cal-event-settings.jpeg"
  />
- Configure o campo com os seguintes valores:
  - **Identificador:** `phone` (tem que ser exatamente assim)

    <img
      height="200"
      src="/resources/cal-phone-field.jpeg"
    />
- Salve as alterações

  <img
    height="200"
    src="/resources/cal-save.jpeg"
  />

### 3. Adicionando o [Cal.com](http://Cal.com) na Assistente

- Vá em **Configurações da Assistente de IA**
- Acesse a seção **Ferramentas (Tools)**
- Adicione a integração com [Cal.com](http://Cal.com)
- Preencha com:
  - Sua chave de API [Cal.com](http://Cal.com)
  - O ID do tipo de evento

    <img
      height="200"
      src="/resources/cal-tool.png"
    />

## Boas Práticas

- Crie tipos de eventos exclusivos para a IA (facilita rastreamento)
- Configure intervalos de buffer entre reuniões
- Defina durações apropriadas
- Use nomes descritivos que façam sentido para o cliente
- Mantenha apenas campos padrão e o de telefone como opcional
- Sempre inclua o e-mail do cliente ao criar o lead

## Testes

Depois de configurar:

- Faça uma chamada de teste para a assistente
- Solicite o agendamento de uma reunião
- Verifique se o agendamento aparece no painel do [Cal.com](http://Cal.com)
- Veja se o número aparece no evento do calendário
- Confirme se o cliente recebeu o e-mail do convite
- Verifique os e-mails de confirmação

## Solução de Problemas

Se houver falhas:

- Verifique se a chave de API está ativa e com permissões corretas
- Confirme se o ID do evento está correto
- Veja se a disponibilidade no [Cal.com](http://Cal.com) está configurada corretamente
- Verifique se não há campos obrigatórios personalizados
- O identificador do campo de telefone deve ser exatamente `phone`
- Confirme se a variável de e-mail está definida na assistente
- Veja se o e-mail do cliente está sendo passado corretamente
- Verifique as configurações de fuso horário
