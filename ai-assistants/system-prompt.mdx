O prompt de sistema é o componente mais importante da sua assistente. Ele define a **personalidade**, **comportamento**, **conhecimento** e **capacidades** da IA. Pense nele como o "cérebro" e o "manual de treinamento" combinados.

## **Comece Rápido com Templates**

Para agilizar a criação:

- Vá nas configurações da assistente
- Encontre o campo **System Prompt**
- Clique em **"Templates"**
- Escolha um modelo conforme seu caso de uso
- Personalize conforme necessário

## **Suporte a Idiomas**

O prompt pode ser escrito em **qualquer idioma**, mesmo que a voz fale outro:

- Escreva o prompt no idioma que preferir
- Defina o idioma falado separadamente nas configurações
- A IA seguirá as instruções do prompt e falará no idioma definido

**Exemplos:**

- Prompt em inglês, voz em espanhol
- Prompt em alemão, voz em francês
- Prompt em chinês, voz em inglês

## **Por que é Importante**

O prompt de sistema:

- Define como a IA pensa e responde
- Delimita os limites da conversa
- Fornece conhecimento essencial
- Controla comportamento e tom de voz
- Determina como agir em situações difíceis

## Componentes Essenciais de um Bom Prompt

A good system prompt should include:

### 1. **Papel e Identidade**

```
Você é um(a) representante de vendas profissional da [Empresa]. Especializa-se em [Produto/Serviço] e tem profundo conhecimento sobre nossas soluções.
```

#### 2. **Estilo de Conversa**

```
Mantenha um tom amigável e profissional. Use linguagem clara e objetiva. Evite jargões técnicos, a menos que o cliente peça.
```

### 3. **Informações-Chave**

```
Nossos principais produtos:
– Produto A (R$X/mês): [características]
– Produto B (R$Y/mês): [características]
Promoção atual: 20% de desconto nos 3 primeiros meses
```

### 4. Diretrizes de Comportamento

```
Verifique os dados do cliente antes de falar sobre a conta

Transfira para um humano se o cliente parecer frustrado

Nunca prometa prazos de entrega
```

### 5. Padrão de Resposta

```
Quando perguntarem sobre preços:

Entenda as necessidades do cliente

Apresente as opções adequadas

Explique o valor de cada pacote

Mencione possíveis descontos
```

## **Boas Práticas**

✅ Seja **específico**\
– Dê instruções claras\
– Inclua exemplos bons e ruins\
– Diga o que **não** deve fazer

✅ Estruture bem\
– Use seções e tópicos\
– Agrupe informações semelhantes

✅ Teste e melhore\
– Faça ligações de teste\
– Refine com base nas gravações\
– Adicione casos de exceção

## **Erros Comuns**

❌ Muito vago: “Seja útil e profissional”\
❌ Muito rígido: Script para todas as falas\
❌ Excesso de informação irrelevante\
❌ Falta de instruções para situações comuns

## Modelo de Estrutura

```
# Papel e Objetivo
[Quem é a IA e o que ela deve fazer]

# Conhecimento Central
[Informações essenciais sobre os serviços/produtos]

# Diretrizes de Conversa
[Como falar com os clientes]

# Padrões de Resposta
[Como responder a perguntas comuns]

# Limites e Restrições
[O que a IA NÃO deve fazer]
```

## **Testes**

Faça chamadas cobrindo:

- Situações comuns
- Casos extremos
- Perguntas difíceis
- Perfis diferentes de clientes

Revise:

- Gravações das ligações
- Precisão das respostas
- Consistência no tom
- Retenção de informações

## **Dicas de Otimização**

- Comece com o **Fast Engine** para testar rápido
- Use gravações para ajustar o prompt
- Inclua exemplos de boas e más respostas
- Prepare a IA para perguntas inesperadas

---

💡 **Dica final:** Seu prompt de sistema é um documento **vivo** – atualize com frequência conforme as ligações reais e feedbacks dos clientes.
