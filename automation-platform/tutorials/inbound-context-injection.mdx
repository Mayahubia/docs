# Injeção de Contexto em Chamadas Recebidas (Inbound Call Context Injection)

Este tutorial demonstra como **aprimorar as conversas do seu assistente de IA** injetando **contexto em tempo real antes da chamada ser conectada**.

## O que você vai aprender

- Configurar o **gatilho de chamadas recebidas (Inbound Call)**
- Buscar os **dados mais recentes de produtos do WooCommerce**
- Estruturar o **contexto para o assistente de IA**

---

## Pré-requisitos

- Conta **Mayahub** com **números de entrada (inbound)**
- **Integração com WooCommerce**
- **Assistente de IA configurado** para injeção de contexto
