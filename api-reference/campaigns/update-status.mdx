---
title: "Atualizar status da campanha"
api: "POST /user/campaigns/update-status"
description: "Iniciar ou parar uma campanha no sistema Autocalls "
---

Este endpoint permite iniciar ou parar uma campanha no sistema MayaHub

### Corpo da requisição

<ParamField body="campaign_id" type="integer" required>
  O ID da campanha a ser atualizada
</ParamField>

<ParamField body="action" type="string" required>
  A ação a ser executada na campanha. Deve ser “start” ou “stop”.
</ParamField>

### Respostas

<ResponseField name="message" type="string" default="Campaign started successfully.">
  A mensagem da resposta
</ResponseField>

<ResponseField name="success" type="boolean" default="true">
  Se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  Os dados da resposta

  <Expandable title="Propriedades dos dados">
    <ResponseField name="campaign_id" type="integer">
      O ID da campanha atualizada
    </ResponseField>
    <ResponseField name="status" type="string" default="in-progress">
      O novo status da campanha (por exemplo, "em andamento" para campanhas iniciadas, "interrompida" para campanhas interrompidas)
    </ResponseField>
  </Expandable>
</ResponseField>
