---
title: "Criar Lead"
api: "POST /user/lead"
icon: "user-plus"
description: "Crie um novo lead no sistema MayaHub"
---

Este endpoint permite criar um novo lead no sistema MayaHub

### Request body

<ParamField body="phone_number" type="string" required>
  Número do telefone no formato E.164 (ex: `+5511999999999`)
</ParamField>

<ParamField body="campaign_id" type="integer" required>
  ID da campanha onde o lead será inserido
</ParamField>

<ParamField body="variables" type="array">
  ID da campanha onde o lead será inserido
</ParamField>

<ParamField body="allow_dupplicate" type="boolean">
  Permitir leads duplicados na mesma campanha (`true` ou `false`)
</ParamField>

### Campos da Resposta (Response Fields)

<ResponseField name="message" type="string">
  Mensagem de retorno da API (ex: "Lead created successfully")
</ResponseField>

<ResponseField name="data" type="object">
  Dados retornados
</ResponseField>

<ResponseField name="data.id" type="string">
  ID único do lead criado
</ResponseField>

<RequestExample>

```json Request
{
  "phone_number": "+5511999999999",
  "campaign_id": 1,
  "variables": [
    {
      "customer_name": "João Silva",
      "email": "joao.silva@exemplo.com"
    }
  ],
  "allow_dupplicate": false
}
```

</RequestExample>

<ResponseExample>

```json 200 Response
{
  "message": "Lead created successfully",
  "data": {
    "id": "1234569"
  }
}
```

</ResponseExample>
