Bem-vindo à documentação da API da MayaHub. Nossa API permite que você interaja programaticamente com a plataforma MayaHub, possibilitando criar e gerenciar leads, agendar chamadas e muito mais.

**URL Base**\
Todas as requisições à API devem ser feitas para a seguinte URL base::

```
https://app.mayahub.online/api
```
