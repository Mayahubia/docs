Bem-vindo à documentação da API da Mayahub.\
Nossa API permite que você interaja programaticamente com a plataforma Mayahub, possibilitando **criar e gerenciar leads**, **agendar chamadas** e muito mais.

## Base URL

All API requests should be made to the following base URL:

```
https://app.mayahub.ai/api
```
