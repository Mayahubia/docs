---
title: "Realizar uma chamada telefônica"
api: "POST /user/make_call"
---

Este endpoint permite realizar uma chamada telefônica.

### Corpo da requisição

<ParamField body="phone_number" type="string" required>
  O número de telefone do cliente no formato E.164 (ex.: +1234567890)
</ParamField>

<ParamField body="assistant_id" type="integer" required>
  O ID do assistente que realizará a chamada
</ParamField>

<ParamField body="variables" type="object">
  As variáveis a serem enviadas para o lead

  <Expandable title="Propriedades das variáveis">
    <ParamField body="customer_name" default="<string>" type="string">
      O nome do cliente
    </ParamField>
    <ParamField body="email" default="<string>" type="string">
      O e-mail do cliente
    </ParamField>
  </Expandable>
</ParamField>

### Respostas

<ResponseField name="message" type="string" default="Call initiated successfully">
  A mensagem da resposta
</ResponseField>

<ResponseExample>

```json 200 Response
{
  "message": "Call initiated successfully",
  "data": {}
}
```

</ResponseExample>
