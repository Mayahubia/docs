A API da MayaHub utiliza chaves de API para autenticar as requisições. Você precisa incluir sua chave de API no cabeçalho de cada requisição feita à API.

## Obtendo uma Chave de API

Para obter uma chave de API:

- Faça login na sua conta MayaHub em [**https://app.**](https://app.autocalls.ai)**mayahub.online**
- Vá até a página “API Keys”
- Clique em “Create new API Key”
- Copie sua nova chave de API e armazene com segurança

<Warning>
  Mantenha sua chave de API confidencial e segura. Não compartilhe publicamente nem envie para sistemas de controle de versão.
</Warning>

## Utilizando a Chave de API

Inclua sua chave no cabeçalho `Authorization` de cada requisição:

```
Authorization: Bearer SUA_CHAVE_DE_API
```

Substitua `SUA_CHAVE_DE_API` pela sua chave real.

## Exemplo de Requisição

Veja um exemplo de como incluir a chave usando cURL:

```bash
curl -X GET "https://api.mayahub.online/api/user/campaigns" \
     -H "Authorization: Bearer SUA_CHAVE_DE_API"
```

Substitua `SUA_CHAVE_DE_API` pela sua chave real ao fazer requisições.
