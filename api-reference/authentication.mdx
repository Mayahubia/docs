A API da Mayahub usa **chaves de API** para autenticar as requisições.\
Você precisa incluir sua **chave de API** no cabeçalho de cada requisição feita à API.

## Obtendo uma Chave de API

Para obter uma chave de API:

- Faça login na sua conta Mayahub em https://app.mayahub.ai
- Acesse a página **"API Keys"**
- Clique no botão **"Create new API Key"**
- Copie sua nova chave de API e armazene-a em local seguro

<Warning>
  Mantenha sua chave de API confidencial e segura.\
  Não a compartilhe publicamente nem a inclua em sistemas de controle de versão.
</Warning>

## Usando a Chave de API

Inclua sua chave de API no cabeçalho **Authorization** de cada requisição:

```
Authorization: Bearer YOUR_API_KEY
```

Substitua **YOUR_API_KEY** pela sua chave de API real.

## Exemplo de Requisição

Abaixo está um exemplo de como incluir a chave de API em uma requisição **cURL**:

```bash
curl -X GET "https://api.mayahub.ai/api/user/campaigns" \
     -H "Authorization: Bearer YOUR_API_KEY"
```

Substitua **YOUR_API_KEY** pela sua chave de API real ao fazer as requisições.
