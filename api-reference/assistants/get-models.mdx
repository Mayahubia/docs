---
title: "Obter modelos LLM disponíveis"
api: "GET /user/assistants/models"
description: "Recuperar todos os modelos LLM disponíveis para configuração do assistente
"
---

Este endpoint retorna uma lista de todos os Modelos de Linguagem de Grande Escala (LLMs) disponíveis que podem ser usados ao criar ou atualizar assistentes.

### Campos de resposta

<ResponseField name="data" type="array">
  <Expandable title="Propriedades">
    <ResponseField name="id" type="integer">
      O identificador único do modelo LLM
    </ResponseField>
    <ResponseField name="name" type="string">
      O nome de exibição do modelo LLM
    </ResponseField>
  </Expandable>
</ResponseField>

### **Observações**

- Este endpoint não requer nenhum parâmetro
- Use o campo **name** ao criar ou atualizar assistentes
- Todos os modelos LLM disponíveis são retornados em uma única solicitação
- Modelos diferentes podem ter capacidades, desempenho e custos variados
