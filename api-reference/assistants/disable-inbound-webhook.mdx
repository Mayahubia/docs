---
title: "Desativar Webhook de Entrada do Assistente"
api: "POST /user/assistants/disable-inbound-webhook"
description: "Desative as notificações de webhook de entrada para um assistente específico"
---

Este endpoint desativa as notificações de webhook de entrada para um assistente, interrompendo a entrega de atualizações em tempo real sobre a conclusão de chamadas recebidas e seus dados.

### Corpo da Requisição

<ParamField body="assistant_id" type="integer" required>
  O ID do assistente para o qual o webhook será desativado
</ParamField>

### Resposta

<ResponseField name="message" type="string">
  Mensagem de sucesso confirmando que o webhook foi desativado
</ResponseField>

<ResponseField name="data" type="array">
  Array vazio (reservado para uso futuro)
</ResponseField>

### Respostas de Erro

<ResponseField name="404 Not Found">
  <Expandable title="Respostas de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro quando o assistente não for encontrado ou não pertencer ao usuário autenticado
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="422 Validation Error">
  <Expandable title="Respostas de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro indicando falha na validação
    </ResponseField>
    <ResponseField name="errors" type="object">
      Erros de validação detalhados para cada campo
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Success Response
{
  "message": "Inbound webhook disabled successfully",
  "data": []
}
```


```json 404 Not Found
{
  "message": "Assistant not found"
}
```


```json 422 Validation Error
{
  "message": "The given data was invalid.",
  "errors": {
    "assistant_id": [
      "The assistant id field is required."
    ]
  }
}
```

</ResponseExample>
