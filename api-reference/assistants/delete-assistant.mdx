---
title: "Deletar um Assistente"
api: "DELETE /user/assistant/{id}"
description: "Deletar um assistente específico"
---

---

Este endpoint permite excluir permanentemente um assistente de IA que pertença ao usuário autenticado.

### **Parâmetros de Caminho**

<ParamField path="id" type="integer" required>
  O identificador único do assistente a ser excluído
</ParamField>

### Resposta

<ResponseField name="message" type="string">
  Mensagem de confirmação indicando que o assistente foi excluído com sucesso
</ResponseField>

### Respostas de Erro

<ResponseField name="404 Not Found">
  <Expandable title="Respostas de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro quando o assistente não for encontrado ou não pertencer ao usuário autenticado
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="500 Internal Server Error">
  <Expandable title="Respostas de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro indicando falha na exclusão do assistente
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Success Response
{
  "message": "Assistant deleted successfully"
}
```


```json 404 Not Found
{
  "message": "Assistant not found"
}
```


```json 500 Internal Server Error
{
  "message": "Assistant deletion failed"
}
```

</ResponseExample>

### **Observações**

- Apenas assistentes que pertencem ao usuário autenticado podem ser excluídos.
- Uma vez excluído, o assistente não pode ser recuperado.
- Esta ação removerá permanentemente o assistente e toda a sua configuração.
- Se o assistente estiver lidando com chamadas ativas, essas chamadas podem ser afetadas.
