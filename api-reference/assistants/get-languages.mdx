---
title: "Obter idiomas disponíveis"
api: "GET /user/assistants/languages"
description: "Recuperar todos os idiomas disponíveis para configuração do assistente
"
---

Este endpoint retorna uma lista de todos os idiomas disponíveis que podem ser usados ao criar ou atualizar assistentes.

### Campos de resposta

<ResponseField name="data" type="array">
  <Expandable title="Propriedades">
    <ResponseField name="id" type="integer">
      O identificador único do idioma
    </ResponseField>
    <ResponseField name="name" type="string">
      O nome de exibição do idioma
    </ResponseField>
    <ResponseField name="iso_2" type="string">
      O código de idioma de duas letras ISO 639-1
    </ResponseField>
  </Expandable>
</ResponseField>

### **Observações**

Este endpoint não requer nenhum parâmetro

Use o campo **name** ao criar ou atualizar assistentes

O código **iso_2** pode ser útil para funcionalidades específicas de idioma

Todos os idiomas disponíveis são retornados em uma única solicitação
