Se você deseja uma **personalização mais avançada** além dos blocos sem código, a [**Mayahub.ai**](http://mayahub.ai) oferece suporte a **webhooks** e **API REST**.

## 1. Webhooks

When to Use: Dispare eventos em sistemas externos no momento em que algo acontece na [Mayahub.ai](http://mayahub.ai) (ex.: "Call Ended,").

Configuration: Defina uma URL de webhook nas configurações do seu assistente/campanha ou na plataforma de automação.

Payload: Normalmente inclui ID da chamada, transcrição, resultado da chamada ou quaisquer dados de usuário coletados.

## 2. REST API

Key Functions:

Criar/Atualizar Leads

Gerenciar Campanhas

Auth & Rate Limits: Normalmente usa tokens de API. Consulte a documentação da API para detalhes sobre endpoints e cotas de uso.

### 3. Casos de Uso Exemplares

Sincronização em Tempo Real com CRM: Para cada nova chamada, envie os dados para o endpoint do seu CRM.

Dashboards Personalizados: Importe dados das chamadas para seu próprio painel de análise.

---

**Observação**: Você encontrará a referência oficial da API em uma seção dedicada da documentação ou pelo painel da [Mayahub.ai.](http://mayahub.ai)\
Sempre proteja seus tokens e teste os endpoints em um ambiente de desenvolvimento.
