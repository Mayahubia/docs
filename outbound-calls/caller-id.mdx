Quando você inicia uma campanha **outbound**, pode escolher **qual número será exibido** no telefone dos destinatários.

## 1. Número Dedicado

- Se você alugou um número local ou gratuito, pode selecioná-lo como **"Phone number"** nas configurações do assistente.
- Ideal para **consistência de marca** ou quando também recebe chamadas **inbound** na mesma linha.
- **Não é necessária verificação por OTP** para números alugados por meio da [**Mayahub.ai**](http://Mayahub.ai).

## 2. Número Existente Verificado (Somente Caller ID)

- Se quiser exibir sua linha pessoal ou da empresa, **verifique-a com um OTP ou chamada curta**.
- Esse número **não atenderá chamadas inbound** da IA, mas o lead verá o número reconhecido no identificador de chamadas.
- **Observação:** a verificação por OTP é **obrigatória apenas** ao usar um número próprio existente como **Caller ID**.

## 3. Integração SIP

- Se você usa uma plataforma **PBX ou VOIP**, pode transmitir um **Caller ID personalizado** do seu sistema existente.
- Certifique-se de que seu **tronco ou provedor** permita definir um Caller ID de saída específico.

## 4. Regras Específicas por País

- Algumas regiões proíbem o uso de números que você não possui como Caller ID.
- Sempre garanta que você tenha o **direito de exibir esse número**, para evitar problemas de **spam** ou **legais**.

**Dica:** teste ligando para o seu próprio telefone ou para um colega antes de lançar uma grande campanha, para confirmar se o **Caller ID correto** está sendo exibido.
