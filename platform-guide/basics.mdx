# Noções Básicas da Mayahub

## Tipos de Chamadas

1. **Chamadas de Entrada:** Chamadas recebidas pelo sistema
2. **Chamadas de Saída:** Chamadas iniciadas pelo sistema

## Campanhas

Campanhas são conjuntos organizados de chamadas com objetivos e públicos-alvo específicos.

## Leads

Leads são potenciais clientes ou contatos que são abordados em uma campanha.

## Configuração do Assistente

- **Mecanismo:** Modelo de IA que alimenta a conversa
- **Prompt:** Instruções iniciais para o assistente de IA
- **Mensagem Inicial:** Primeira mensagem falada pelo assistente
- **Idioma:** Idioma utilizado na conversa
- **Voz:** Voz selecionada para o assistente
- **Ferramentas:** Funcionalidades adicionais disponíveis para o assistente
- **Extrações Pós-Chamada:** Dados extraídos após a chamada para análise

## Plataforma de Automação

Uma plataforma **no-code** para configurar e gerenciar chamadas automatizadas.

## Provisionamento de Números de Telefone

Processo de aquisição e configuração de números de telefone para uso em campanhas.

## Custos

- **Custo por minuto da Mayahub:** Cobrança pelo uso do serviço de chamadas automatizadas
- **Custo Twilio:** Taxas adicionais pelo uso dos serviços de comunicação da Twilio

## Ferramentas

- **Transferir Chamada:** Capacidade de transferir a ligação para um atendente humano
- **Encerrar Chamada:** Opção para finalizar a ligação
- **Integração com **[**Cal.com**](http://Cal.com)**:** Funcionalidade de agendamento integrada ao Cal.com
